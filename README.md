## 📖 Project Description

TaskFlow Pro is a client-side To-Do List application designed to demonstrate JavaScript DOM manipulation, event handling, state management, and browser storage techniques.

Users can create, update, delete, and manage tasks effortlessly. The application stores task data using the browser's Local Storage API, ensuring tasks remain available even after refreshing or reopening the browser.

The project features a premium glassmorphism interface with animations and responsive layouts, providing an engaging user experience across devices.
